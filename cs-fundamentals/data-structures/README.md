## 💡 How to Study Data Structures

For each data structure, understand:
1. **What is it?** — The concept and mental model
2. **How does it work?** — Internal mechanics
3. **Time & Space Complexity** — Big O notation
4. **When to use it?** — Real-world use cases
5. **Code it** — Implement from scratch in Python or JS
