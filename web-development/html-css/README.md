## 💡 Quick Tips

- Always use semantic HTML for better accessibility and SEO
- Use CSS custom properties (`--color-primary`) for maintainable styles
- Mobile-first approach: style for small screens first, then scale up
- Use `box-sizing: border-box` for predictable sizing
