## 💡 Prerequisites

Before starting React, make sure you're comfortable with:
- ✅ HTML & CSS fundamentals
- ✅ JavaScript ES6+ (arrow functions, destructuring, spread/rest)
- ✅ Array methods: `map`, `filter`, `reduce`
- ✅ Promises and async/await
