## 💡 MongoDB vs SQL Quick Comparison

| SQL | MongoDB |
|-----|---------|
| Table | Collection |
| Row | Document |
| Column | Field |
| JOIN | $lookup (aggregation) |
| Primary Key | _id (ObjectId) |
| Schema (fixed) | Schema (flexible) |
