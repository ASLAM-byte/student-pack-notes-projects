## 💡 SQL Quick Reference

```sql
-- Select all records
SELECT * FROM table_name;

-- Filter records
SELECT * FROM users WHERE age > 18;

-- Insert a record
INSERT INTO users (name, email) VALUES ('Ayaan', 'ayaan@example.com');

-- Update a record
UPDATE users SET name = 'Ayaan K' WHERE id = 1;

-- Delete a record
DELETE FROM users WHERE id = 1;
```
