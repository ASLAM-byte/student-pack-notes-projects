## 🐙 Git Tips

- Write commit messages in present tense: `"Add login feature"` not `"Added"`
- Commit often — small commits are easier to debug than large ones
- Always `git pull` before starting new work to avoid merge conflicts
- Use `.gitignore` to prevent sensitive files from being committed

---

## 🐍 Python Tips

- Use f-strings for string formatting: `f"Hello, {name}!"`
- List comprehensions are faster and more Pythonic than loops for simple tasks
- `enumerate()` gives you both index and value when looping
- Use `if __name__ == "__main__":` to make scripts importable as modules

---

## 🌐 Web Dev Tips

- Always validate HTML at [validator.w3.org](https://validator.w3.org)
- Use browser DevTools (F12) to debug CSS and JavaScript
- Start mobile-first: design for small screens, then expand
- Use `rem` instead of `px` for font sizes — respects user preferences
