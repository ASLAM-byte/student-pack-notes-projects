## 📝 How I Take Book Notes

For each chapter I read, I note:
- **Main idea** — What's the core concept?
- **Key examples** — Code or illustrations that clicked
- **My takeaway** — How will I apply this?
- **Page references** — For re-reading later

## 💡 Finding Free Books

- Many classic programming books are freely available online
- Check [GitHub Student Pack](https://education.github.com/pack) for book deals
- Public libraries often have free access to O'Reilly, Packt, etc.
