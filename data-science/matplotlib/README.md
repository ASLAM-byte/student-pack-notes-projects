## 💡 Quick Start

```python
import matplotlib.pyplot as plt

# Simple line chart
x = [1, 2, 3, 4, 5]
y = [2, 4, 6, 8, 10]

plt.plot(x, y, marker='o')
plt.title('My First Chart')
plt.xlabel('X Axis')
plt.ylabel('Y Axis')
plt.show()
```
