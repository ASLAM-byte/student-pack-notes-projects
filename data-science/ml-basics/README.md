## 💡 ML Workflow

1. **Define the problem** — What are you predicting?
2. **Collect & clean data** — pandas + data wrangling
3. **Explore the data** — visualizations, statistics
4. **Choose a model** — start simple (linear regression)
5. **Train the model** — `model.fit(X_train, y_train)`
6. **Evaluate the model** — accuracy, RMSE, F1 score
7. **Improve** — tune hyperparameters, try new features
