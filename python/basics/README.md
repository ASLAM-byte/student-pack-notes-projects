## 💡 Python Tips for Beginners

- Python uses **indentation** (spaces/tabs) instead of `{}` for code blocks
- Use `#` for single-line comments and `"""..."""` for multi-line
- Python is **dynamically typed** — no need to declare variable types
- The `print()` function is your best debugging friend!
- Run Python files with: `python hello.py` or `python3 hello.py`
